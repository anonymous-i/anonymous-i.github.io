This project is a text summarization project in the NLP (Natural language processing) domain, which tries to combine the extractive and abstractive based methods for text summarization. The extractive based text summarization is a text-rank / sentence scoring based task while the abstractive based task uses the deep learning approach using LSTM networks to generate summary sequences. The project also augments the topic modelling task using SVM classification of the document to identify a topic such as Crime, Enivironment, News, Health and 40 such topics. The project then also augments the task of Entity extraction using POS tagging and Named Entity Recognition (NER) to identify real time entities present in the document such as person, organization, country etc.
Project Repo : Github

The Extractive text summarization project tries to generate summary of an input text document/ web articles using the NLP based sentence scoring feature extraction and K-Means clustering approach.
Project Repo : Github

This project aims at the implementation of two different queue based spin locks, which are proposed in the literature as new advancements to the classical, naïve queue based spin locks called “Ticket Locks”. These naïve locks are prevalent in the Red hat linux kernels since 2015 and were first introduced to the system in the year 2008. In this project we basically implement a certain newer advanced versions of these naive locks to overcome certain limitations. We look at the Partitioned locks proposed by David Dice. 2011 and the TWA locks proposed by Dave Dice, Oracle Labs ,Alex Kogan,Oracle Labs. We implement these locks and analyse the performance statistics of these locks across a set of different parameters.
Project Repo : Github

We implement the HVAC system by emulating the system in real time , we emulate the various devices and the thermostat sensor that senses the ambient temperature in the indoor environment. Our system’s focus is to maintain the equilibrium thermal stability in the environment by regulating the temperatures using appliances such as the Air Conditioner, Fan and the Room Heater. All the appliances in our project are centrally co-ordinated by a higher level device called the “Central Hub” or the gateway that takes the decisions for temperature regulations and also establishes the communication among the devices.
